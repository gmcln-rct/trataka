ephemeralLight

ephemeralLight is based on the tantric meditation method of Trāṭaka ("look" or "gaze") which involves staring at a single point, such as a small object or candle light, to gain focus and achieve mindfulness. Trāṭaka is often used for yogic purification.

This project uses the Tone.js JavaScript library to generate a tonal soundscape each time the button is pressed. Each individual soundscape -- and the flame audio-visualization it creates -- is unique and ephemeral.

Created by: Glenn McClanan

Technologies: JavaScript & Tone.js

Best experienced with headphones.
(Audio optimized for Chrome desktop/laptop, disabled on iOS.)
